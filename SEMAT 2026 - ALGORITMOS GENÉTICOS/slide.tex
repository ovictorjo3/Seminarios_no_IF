	\section{Introdução}
	
		\begin{frame}{Introdução}
			\justifying
		
			O Problema da Mochila e a minimização da função $f(x)=x(\sin(10\pi x))+1$ são problemas de otimização resolvidos com Algoritmos Genéticos (AG). O primeiro busca maximizar o valor de itens com peso limitado. O segundo encontra o mínimo global de uma função com múltiplos mínimos locais. Em ambos, o AG encontra soluções próximas da ótima de forma eficiente, sem exigir derivadas ou buscas exaustivas.
		\end{frame}
	
	
		\begin{frame}{Definição}
			\begin{block}{Conforme Pozo et al. (2003): }
				\begin{itemize}
					\item O Algoritmo Genético é uma das técnicas da computação evolucionária de busca e otimização inspirada na evolução natural das espécies de Darwin.
				
					\item \uncover<2->{A ideia é criar uma população de indivíduos que vão se reproduzir e competir pela sobrevivência, onde os melhores indivíduos sobrevivem e transferem suas características para novas gerações, até que se alcance uma solução próxima ao resultado ótimo.} 
				\end{itemize}
			\end{block} 
		\end{frame}
	
		\begin{frame}{O Algortimo Genético}
			\justifying
			Para Vitor Emanuel Batista (2020), o desenvolvimento de simulações computacionais de sistemas genéticos teve início na década de 50 e 60 a partir de biólogos e foi John Holland quem iniciou as primeiras pesquisas sobre o tema, publicando, em 1975, o livro “Adaptation in Natural and Artificial Systems”.
		\end{frame}
	
		\begin{frame}{O Algortimo Genético}
			\justifying
			Posteriormente se iniciariam aplicações industriais dos algoritmos genéticos com David E. Goldberg, aluno de Holland, e hoje esses algoritmos são utilizados para solucionar problemas de otimização e aprendizado de máquinas, argumenta Batista (2020).
		\end{frame}
	
		\begin{frame}{O Algortimo Genético}
			\justifying
			Conforme Pozo et al. (2003), a ideia básica de funcionamento dos algoritmos genéticos é a de tratar as possíveis soluções do problema como "indivíduos" de uma "população", que irá "evoluir" a cada iteração ou "geração". Para isso é necessário construir um modelo de evolução onde os indivíduos sejam soluções de um problema
		\end{frame}
	
		\begin{frame}{O Algortimo Genético}
			\begin{itemize}
			\item Antes de detalhar o comportamento do algoritmo genético propriamente dito, se faz necessária a compreensão da terminologia adotada para representar as entidades e processos envolvidos. 
			
			\uncover<2->{
			\item Os termos, análogos aos da biologia, referem-se no algoritmo à construção das possíveis soluções ao problema abordado.
			}
			\end{itemize}
		\end{frame}
	
	\section{Terminologia de um AG}
	
		\begin{frame}{Terminologia de um AG}
			Vamos detalhar os seguintes termos da imagem:
		
			$$\includegraphics[scale=0.35]{imagens/terminologia.png}$$
		\end{frame}
		
		\begin{frame}{Cromossomo}
			\begin{block}{Conforme Koza (1992) e Batista (2020):}
				\begin{itemize}
					\justifying
					\item Cada cromossomo representa uma solução do problema, tomado também como sinônimo de indivíduo, representado por uma cadeia de símbolos (genes).
					
					\uncover<2->{
					\item Uma das principais formas de representar um cromossomo é através de uma sequência de bits, onde cada bit ativo (1) representa a presença de uma característica e cada bit inativo (0) representa a ausência dessa característica 
					}
				\end{itemize}
			\end{block}
		\end{frame}
		
		\begin{frame}{Cromossomo}
			\justifying
			
			Comparando com a genética dos seres vivos, cada bit da cadeia representaria um gene, menor unidade de informação em um cromossomo, e a cadeia de bits representa o \textbf{cromossomo}, que é a resultante do problema.
		\end{frame}
		
		\begin{frame}{População}
			\begin{block}{Segundo Patricia Vargas (2000:)}
				\justifying
				\begin{itemize}
					\item É o conjunto de cromossomos (ou indivíduos) gerados que são disponibilizados como possíveis soluções para o problema apontado e que serão utilizados para a produção de novos indivíduos;
					
					\uncover<2->{
					\item O tamanho da população inicial e das subsequentes gerações são um aspecto importante a ser considerado para o sucesso do processo evolutivo, pois é possível comprometer a evolução dos indivíduos caso a quantidade escolhida seja muito pequena ou muito grande.
					}
				\end{itemize}
			\end{block}
		\end{frame}
		
		\begin{frame}{Cruzamento (ou \textit{crossover})}
			\begin{block}{Fabrício Bueno (2009) argumenta:}
				\begin{itemize}
					\item É um processo de reprodução sexuada em que há combinação de genes dos cromossomos originando um ou mais descendentes;
					
					\uncover<2->{
					\item O cruzamento é o principal responsável pela variabilidade genética.
					}
				\end{itemize}
			\end{block}
		\end{frame}
		
		\begin{frame}{Cruzamento (ou \textit{crossover})}
			\justifying
			Exemplo de reprodução sexuada em que há combinação de genes dos cromossomos originando um ou mais descendentes. 
			
			$$\includegraphics[scale=0.3]{imagens/crossover.png}$$
			
		\end{frame}
		
		\begin{frame}{Outras terminologias}
			\begin{block}{Fabrício Bueno (2009) apresenta:}
				\justifying
				\begin{itemize}
				
					\item \textbf{Mutação}: anomalias que causam a alteração aleatória de genes, seja na sua localização, seja no seu conteúdo;
					
					\uncover<2->{
					\item \textbf{Geração}: iteração do algoritmo genético;
					}
					
					\uncover<3->{
					\item \textbf{Aptidão (ou fitness)}: indicador qualitativo de um indivíduo. O grau de aptidãode um indivíduo é obtido a partir de uma função objetivo;
					}
					
					\uncover<4->{
					\item \textbf{Função objetivo}: função matemática que avalia as soluções (indivíduos) emrelação ao problema.
					}
				\end{itemize}
			\end{block}
		\end{frame}
		
	\section{Conversão binário-decimal}
	
		\begin{frame}{Conversão binário-decimal}
			\justifying
			Você já sabe o que é um cromossomo, população e fitness. Mas como um AG resolve o problema da função $f(x)=x(\sin(10\pi x))+1$ se ele trabalha com bits? A resposta está na conversão de binário para decimal.
		\end{frame}
		
		\begin{frame}{Conversão binário-decimal}
			\begin{block}{Paulo Gabriel e Alexandre Delbem (2008) \textit{apud} Goldberg (1989)}
				\justifying 
				Nos AGs canônicos (Goldberg, 1989), as soluções candidatas são codificadas em arrays
				binários de tamanho fixo. A motivação para o uso de codificação binária é oriunda da teoria dos
				esquemas (Holland, 1992) utilizada com sucesso para explicar por que os algoritmos genéticos
				funcionam.
			\end{block}
		\end{frame}
		
		\begin{frame}{Método das divisões sucesivas}
			\justifying
			Conforme Capuano e Ideota (1997), consiste em efetuar-se sucessivas divisões pela base a ser convertida (no caso o 2) até o último quociente possível. O número transformado será composto por este último quociente (algarismo mais significativo) e, todos os restos, na ordem inversa às divisões.
		\end{frame}
